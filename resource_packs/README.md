# Resource Packs

Place Minecraft Java-compatible resource pack ZIPs or folders here.

Examples:
- `Faithful-32x-1.21.11.zip`
- `Faithful-32x-1.21.11/` (extracted folder)
- `MyCustomPack/`

## Required structure

```
pack.mcmeta
assets/minecraft/textures/block/*.png
```

## Usage

Start the game, open Settings → Texture Packs, select a pack and press Apply.
The pack is applied without restarting. Missing textures fall back to the built-in defaults.
